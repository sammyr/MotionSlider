Hier kommen die Icon-Dateien für die Titelleiste und das Programm-UI hinein.
Beispiel: appicon.ico, testicon1.png, testicon2.ico etc.
Icons können frei aus dem Netz geladen werden (z.B. von https://icons8.com oder https://iconarchive.com).

Für das Explorer-Icon sollte eine .ico-Datei zusätzlich im Hauptverzeichnis liegen (z.B. appicon.ico).
